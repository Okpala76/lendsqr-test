# lendsqr-test

An admin dashboard for managing users, loans, and financial data.

## Overview

**lendsqr-test** is a modern admin dashboard built as an assessment project.  
It allows internal teams and developers to view, filter, and manage user data through a clean and responsive interface.

Although built as an assessment, the app follows production-ready patterns and is deployed publicly.

## Key Features

- View and filter users in a paginated table
- User details page with structured information
- User status management (Active / Blacklisted)
- Responsive dashboard layout with sidebar and header

## Tech Stack

- **Next.js** (App Router)
- **React Query** for data fetching and caching
- **TanStack Table** for flexible table rendering
- **SCSS Modules** for scoped styling
- **pnpm** as the package manager

## Data Source

This project uses a **Mock API** to simulate user data and interactions.

## Running Locally

```bash
pnpm install
pnpm run dev
```
